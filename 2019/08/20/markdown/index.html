二.字体

加粗

加粗文字左右用两个*号包起来
示例:
我加粗了

斜体

要倾斜的字体分别左右用一个号包起来
示例：
*我倾斜了

斜体加粗

要倾斜加粗的字体分别左右用三个号包起来
示例：
**我倾斜加粗了***

删除线

要删除线的字体分别左右用两个号包起来
示例：
~我加删除线了~~

八.表格

表头|表头|表头
-–|:–:|—:
内容|内容|内容
内容|内容|内容
第二行分割表头和内容。
- 有一个就行，为了对齐，多加了几个
文字默认居左
-两边加：表示文字居中
-右边加：表示文字居右
注：原生的语法两边都要用 | 包起来。此处省略
示例：

姓名 技能 排行
刘备 哭 大哥
关羽 打 二哥
张飞 骂 三弟

九.代码

单行代码：代码之间分别用一个反引号包起来
print(np.shape(w),np.shape(h))
代码块：代码之间分别用三个反引号包起来，且两边的反引号单独占一行
